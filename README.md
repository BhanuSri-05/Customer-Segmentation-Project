# Customer Segmentation Using K-Means Clustering

## Project Overview

This project segments retail customers based on purchasing behavior and demographics using K-Means Clustering.

The objective is to identify different customer groups and provide actionable business insights for targeted marketing and customer retention.

---

## Problem Statement

Customers have different spending habits and income levels. Treating all customers the same can reduce marketing effectiveness.

This project groups customers into meaningful segments based on:

* Annual Income
* Spending Score
* Customer Demographics
